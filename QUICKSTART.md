# 快速开始指南 (Quick Start Guide)

## 5分钟上手股票监控系统

### 第一步: 安装依赖

```bash
pip install -r requirements.txt
```

### 第二步: 配置自选股

编辑 `config.json` 文件，添加您要监控的股票代码:

```json
{
  "watchlist": [
    "600000",  // 浦发银行
    "000001",  // 平安银行  
    "600519"   // 贵州茅台
  ],
  "scan_interval": 60
}
```

### 第三步: 运行程序

#### 方式1: 图形界面（推荐）

```bash
python stock_monitor.py
```

点击"启动监控"按钮即可开始监控。

#### 方式2: 命令行模式

```bash
python stock_monitor.py --cli
```

按 Ctrl+C 停止监控。

#### 方式3: 快速测试

```bash
python demo.py
```

执行一次扫描，查看效果。

### 第四步: 查看提醒

当检测到以下情况时，程序会自动弹窗提醒：

✅ **价格异动**: 涨跌幅超过3%（可配置）  
✅ **买入信号**: 综合评分≥5分  
✅ **卖出信号**: 综合评分≤-5分  
✅ **重要新闻**: 检测到重大公告

## 技术指标说明

程序使用多种技术指标综合分析：

| 指标 | 说明 | 信号 |
|------|------|------|
| **MA** | 移动平均线 | 多头排列=买入，空头排列=卖出 |
| **RSI** | 相对强弱指标 | <30超卖(买)，>70超买(卖) |
| **MACD** | 指数平滑移动平均线 | 金叉=买入，死叉=卖出 |
| **KDJ** | 随机指标 | J<20超卖(买)，J>80超买(卖) |

## 高级配置

### 调整扫描间隔

```json
{
  "scan_interval": 30  // 改为30秒扫描一次
}
```

### 调整提醒阈值

```json
{
  "alert_conditions": {
    "price_change_threshold": 2.0,     // 2%变动就提醒
    "buy_signal_threshold": 3,         // 评分≥3就买入提醒
    "sell_signal_threshold": -3        // 评分≤-3就卖出提醒
  }
}
```

## 常见问题

### Q1: 没有tkinter怎么办？

A: 使用命令行模式：`python stock_monitor.py --cli`

### Q2: 如何添加更多股票？

A: 编辑 `config.json` 的 `watchlist` 数组，添加股票代码。

### Q3: 如何修改API接口？

A: 编辑 `stock_monitor.py` 中的 `TongHuaShunAPI` 类，替换为真实API。

### Q4: 程序能24小时运行吗？

A: 可以！使用CLI模式在服务器后台运行：
```bash
nohup python stock_monitor.py --cli > monitor.log 2>&1 &
```
